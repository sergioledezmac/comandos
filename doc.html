Comandos Linux FTP

              
                Conectarse a un servidor FTP desde la terminal de linux
                ftp 192.168.0.101
                O
                ftp root@192.168.0.101
                
                Descargar un Archivo desde el FTP con el comando GET
                get backup.zip
                
                Especificar el Directorio de descarga desde el FTP al Equipo Local con el comando LCD
                lcd ~/Documentos
                
                Descargar varios archivos a la vez, con el comando MGET
                mget backup1.zip backup2.zip
                
                Cargar un archivo desde un directorio local a un servidor FTP comando PUT
                put image.png
                Utilizar la ruta si el archivo no se encuentra en el directorio actual put /home/sergio/pictures/image.png
                
                
                Cargar múltiples archivos desde un directorio local a un servidor FTP comando MPUT
                mput image1.png image2.png
                
                Cerrar la conexión FTP con BYE o QUIT
                
                Para conocer más opciones usar el comando MAN FTP
